## 1. Типы счетов для нерезидентов

Нерезиденты (физические и юридические лица, зарегистрированные за пределами Республики Сербия) могут открыть следующие типы счетов:

### 1.1 Текущие счета (текући рачун)

| Тип счёта | Валюта | Описание |
|-----------|-------|----------|
| Динарский текущий счёт (динarski текући рачун) | RSD (сербский динар) | Основной счёт для операций в Сербии |
| Валютный текущий счёт (девизни текући рачун) | EUR, USD, CNY, RUB, CHF, GBP, SEK, NOK, CAD, AUD | Мультивалютный счёт |

### 1.2 Специальные счета

- **Динарский счёт для специальных целей** (девизни рачун за посебне намене) — для средств, которые по закону должны храниться отдельно
- **Валютный счёт для специальных целей** — аналогично для валюты

## 2. Документация для открытия счёта

### 2.1 Основные формы (обязательные)

Скачать с сайта банка: https://www.posted.co.rs/privreda/platni-racuni/dinarski-devizni-racun-nerezidenti.html

| Форма № | Название (сербский/английский) | Описание |
|--------|--------------------------------|----------|
| TRP-501 | Zahtev za otvaranje racuna pravnog lica / Application for opening account | Заявление на открытие счёта юридического лица/ИП |
| TRP-507 | Karton deponovanih potpisa / Authorized signatures sheet | Карточка образцов подписей |
| TRP-510 | Podaci o osnicacima pravnog lica / Data on founders | Данные об учредителях |
| TRP-511 | Podaci o osnicacima osnicava / Data on founders of founders | Данные об учредителях учредителей |
| — | Informacija o obradi podataka / Information on data processing | Информация об обработке персональных данных |

### 2.2 Дополнительные документы для нерезидентов

**Для юридических лиц (нерезидентов):**

1. **Выписка из регистра** (Извод из Регистра) в стране регистрации, не старше 3 месяцев
   - Или иной соответствующий документ, если в стране регистрации нет такого регистра

2. **Документ с налоговым идентификационным номером (ПИБ/TIN)** — от налоговых органов страны регистрации

3. **Для котируемых акционерных обществ:**
   - Документ о нахождении на бирже
   - Указание биржи, на которой осуществляется котировка

4. **Если учредитель — другое юридическое лицо с долей ≥25%:**
   - Выписка из регистра с данными до конечных бенефициаров (физических лиц), не старше 6 месяцев

5. **Если учредитель — иностранное юридическое лицо:**
   - Решение о регистрации/Выписка из регистра, не старше 6 месяцев, заверенные нотариусом или судом в соответствии с правилами международной легализации

6. **Если в выписке из регистра не видно реального собственника:**
   - Выписка из документации, не старше 6 месяцев, с данными о конечных бенефициарах

7. **Перевод решения о регистрации** иностранного юридического лица
   - Перевод заверен присяжным переводчиком на сербский язык

8. **Доверенность** (Пуномоћје), если счёт открывает доверенное лицо
   - Заверена нотариусом

9. **Копия документа, удостоверяющего личность**
   - Законного представителя/учредителя/доверенного лица

### 2.3 Для физических лиц-нерезидентов

- Паспорт или иной документ, удостоверяющий личность
- Документ, подтверждающий место жительства (не старше 3 месяцев)
- Налоговый номер (TIN) или аналог из страны резиденства

### 4.1 Доступные услуги

| Услуга | Описание | Доступность для нерезидентов |
|--------|---------|------------------------------|
| **Homebanking (e-Banking)** | Интернет-банкинг | Доступен |
| **Mobile banking** | Мобильное приложение | Доступно |
| **SMS уведомления** | SMS-оповещения | Только для сербских мобильных операторов |

### 4.2 Функции e-Banking

- Управление счетами
- Переводы и платежи
- Просмотр выписок
- Оплата услуг
- Управление лимитами карт

### 4.3 Регистрация

Регистрация в e-Banking осуществляется:
1. **В отделении банка** — при открытии счёта
2. **Через систему Homebanking** — после получения учетных данных

## 5. Пошаговая инструкция

### Шаг 1: Подготовка документов

1. Соберите все необходимые документы (см. раздел 2)
2. Сделайте нотариально заверенные переводы документов на сербский язык (при необходимости)
3. Получите документы с апостилем или нотариальным заверением (если требуется)

### Шаг 2: Скачайте и заполните формы

1. Перейдите на страницу: https://www.posted.co.rs/privreda/platni-racuni/dinarski-devizni-racun-nerezidenti.html
2. Скачайте формы TRP-501, TRP-507, TRP-510, TRP-511
3. Заполните формы на сербском или английском языке

### Шаг 3: Посетите отделение банка

1. Найдите ближайшее отделение: https://www.posted.co.rs/o-nama/ekspoziture.html
2. Лично подайте заявление
3. Представьте все документы
4. Пройдите идентификацию

### Шаг 4: Открытие счёта

1. Банк рассматривает заявление
2. При положительном решении счёт открывается
3. Вы получаете номер счёта и доступ к e-Banking

### Шаг 5: Получение карты

1. Подайте заявление на выпуск карты в отделении
2. Дождитесь изготовления карты (обычно 5-10 рабочих дней)
3. Получите карту лично в отделении

```
Вид счёта (одно):
☐ Динарский текущий счёт
☐ Валютный текущий счёт (выбрать валюту: ___)

Тип клиента: ☑ Нерезиденты (Non-residents)

Данные о клиенте:
- Бизнес-наименование: [название компании]
- Адрес офиса: [адрес в стране регистрации]
- Матичный номер: [регистрационный номер]
- ПИБ (TIN): [налоговый номер]

Контактное лицо: [ФИО]
Контактный телефон: [телефон]
Email: [email]
```

### Форма TRP-510 — Данные об учредителях

Заполняется для каждого учредителя с долей ≥25%:
- Название/ФИО
- Регистрационный номер / JMBG
- Адрес
- Дата рождения
- Гражданство
- Номер паспорта
- Процент владения
